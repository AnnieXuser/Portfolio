# 🎵 Pre-dream: 你的睡前故事与音乐伴侣

Pre-dream​ 是一个基于 Python 的互动式聊天机器人，专为睡前放松和冥想设计。它通过对话了解你的偏好，为你生成个性化的睡前故事，并推荐与心情相匹配的歌曲，帮助你在音乐和故事的陪伴中平静入睡。
